== Description ==

Allows adding a role to a user who has earned a BadgeOS achievement.

**Note:** You will need to install the free [BadgeOS plugin](http://wordpress.org/extend/plugins/badgeos/ "BadgeOS")&trade; (version 3.0 or higher) to use the BadgeOS Community Add-on. This Plugin was tested up to BadgeOS 3.6.7.

[Get the BadgeOS plugin](http://wordpress.org/extend/plugins/badgeos/ "BadgeOS").

This plugin was originally developed for the "Globe - Community of Digital Learning" on behalf of [GIZ](https://www.giz.de/).

== Installation ==

1. Upload, activate and configure the free [BadgeOS plugin](http://wordpress.org/extend/plugins/badgeos/ "BadgeOS") to WordPress.
2. Upload 'BadgeOS-Award-Role-Add-on' to the '/wp-content/plugins/' directory
3. Activate the plugin through the 'Plugins' menu in WordPress
